To view GO documentation locally in a web browser:

>Type the following:  $godoc -http=: 8999
>
>Open up a browser and go to the URL: http://localhost:8999

Install Git, Mercurial and Bazaar
$brew install git
$brew install mercurial
$brew install bazaar

Check on installed
$brew info git
$brew info mercurial
$brew info bazaar
